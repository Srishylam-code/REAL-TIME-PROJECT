# AI Crop Recommendation System v4.0 (Full-Stack AI)

This project has been upgraded to a professional **Full-Stack AI/ML Architecture** to meet academic and industry standards.

## 🧠 Architecture Overview
The system now uses a multi-layered intelligence approach:
1.  **Frontend**: Interactive cinematic UI (HTML/CSS/JS).
2.  **ML Model (Python)**: A **Random Forest Classifier** trained to predict optimal crops based on NPK, pH, and climate data.
3.  **Online Expert AI**: Integrated **Google Gemini Pro** to provide real-time agricultural advisory for the predicted crops.
4.  **Live Weather Data**: Real-time environmental parameters via **Open-Meteo API**.

## 🛠️ Technical Implementation
### 1. Python Backend (`backend/app.py`)
- Built with **Flask**.
- Implements a **Random Forest** algorithm using `scikit-learn`.
- Handles data preprocessing and online AI orchestration.
- Connects to Google's Generative AI via `google-generativeai`.

### 2. JavaScript Integration (`modules/api.js`)
- Communicates with the Python ML server via REST API.
- Implements a "Reliability Fallback" mechanism: If the ML server is down, it uses a local heuristic engine to ensure the platform remains functional.

### 3. Comparison & Calculator
- **Profit Scalability**: Logic for calculating total income based on land area.
- **Side-by-Side Comparison**: Technical table comparing ML predictions.

## 🚀 How to Run for Faculty Demo
1.  **Install Python Dependencies**:
    ```bash
    pip install -r backend/requirements.txt
    ```
2.  **Start Python AI Server**:
    ```bash
    python backend/app.py
    ```
3.  **Open the Platform**:
    Run `server.js` or open `index.html` via Live Server.

4.  **Demonstrate "Online Output"**:
    When you enter soil data, the system will show **"(AI Recommended)"** next to the crop name, proving it is being generated by the Machine Learning model.
