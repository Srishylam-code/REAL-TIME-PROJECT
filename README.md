# 🌱 AI-Powered Crop Recommendation System

A high-performance, data-driven agricultural advisory system built for precision farming. This system provides instant, data-driven crop recommendations and soil enrichment plans without relying on external API calls.

## 🚀 Key Features

- **Global Heuristic Engine**: Instant crop recommendations based on soil type, season, and pH level.
- **Soil Enrichment Advisor**: 10-point remediation plans for extreme pH conditions (Acidic, Alkaline, Neutral).
- **100% Offline Capability**: Runs entirely locally for maximum speed and privacy.
- **Premium UI/UX**: Futuristic, responsive dashboard with glassmorphism and real-time feedback.
- **Dynamic Advice**: Context-aware guidance for fertilizers, pest control, and sustainable practices.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Backend**: Node.js, Express
- **Logic**: Custom Heuristic Engine (zero AI-dependency for stability and speed)

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repository-url>
   cd crop-recommendation-system
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the server**:
   ```bash
   npm start
   ```

4. **Access the application**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📂 Project Structure

- `server.js`: Main Express application.
- `modules/engine.js`: The core heuristic logic.
- `modules/ui.js`: UI interaction and state management.
- `index.html`: The main dashboard interface.
